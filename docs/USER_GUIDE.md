# 📖 Guide Utilisateur - FinanzPlus Austria

## Bienvenue sur FinanzPlus Austria

FinanzPlus Austria est votre plateforme de confiance pour comparer et obtenir les meilleures offres de crédit en Autriche. Ce guide vous aidera à utiliser toutes les fonctionnalités de la plateforme.

---

## 🚀 Démarrage Rapide

### 1. Créer un Compte

1. Cliquez sur **"Registrieren"** dans le menu
2. Remplissez le formulaire :
   - Prénom et Nom
   - Adresse email
   - Mot de passe (min. 8 caractères)
   - Numéro de téléphone
3. Acceptez les conditions d'utilisation
4. Cliquez sur **"Konto erstellen"**
5. Vérifiez votre email pour confirmer votre compte

### 2. Se Connecter

1. Cliquez sur **"Anmelden"**
2. Entrez votre email et mot de passe
3. Cliquez sur **"Anmelden"**

---

## 💰 Simulateur de Prêt

### Utiliser le Simulateur

1. Accédez à **"Kreditrechner"** dans le menu
2. Renseignez les informations :
   - **Montant** : Entre 1 000€ et 500 000€
   - **Durée** : Entre 12 et 360 mois
   - **Objectif** : Sélectionnez le type de prêt
3. Cliquez sur **"Berechnen"**

### Comprendre les Résultats

Le simulateur affiche :
- **Mensualité** : Montant à payer chaque mois
- **Coût total** : Montant total à rembourser
- **Intérêts totaux** : Coût du crédit
- **Taux d'intérêt** : Taux annuel effectif global (TAEG)

### Tableau d'Amortissement

Le tableau détaille pour chaque mois :
- Numéro du mois
- Mensualité
- Part d'intérêts
- Part du capital
- Capital restant dû

### Exporter en PDF

1. Cliquez sur **"Als PDF exportieren"**
2. Le fichier se télécharge automatiquement
3. Vous pouvez l'imprimer ou le sauvegarder

---

## 📝 Faire une Demande de Prêt

### Étape 1 : Simuler votre Prêt

1. Utilisez le simulateur pour calculer votre prêt
2. Vérifiez que les conditions vous conviennent
3. Cliquez sur **"Diesen Kredit beantragen"**

### Étape 2 : Remplir le Formulaire

Informations requises :
- **Informations personnelles**
  - Prénom, Nom
  - Date de naissance
  - Adresse complète
- **Informations professionnelles**
  - Situation professionnelle
  - Employeur
  - Revenu mensuel net
- **Informations du prêt**
  - Montant souhaité
  - Durée
  - Objectif du prêt

### Étape 3 : Télécharger les Documents

Documents nécessaires :
- ✅ Pièce d'identité (passeport ou carte d'identité)
- ✅ Justificatif de domicile (< 3 mois)
- ✅ 3 derniers bulletins de salaire
- ✅ Relevés bancaires (3 derniers mois)

**Format accepté** : PDF uniquement  
**Taille maximale** : 5 MB par fichier

### Étape 4 : Soumettre la Demande

1. Vérifiez toutes les informations
2. Acceptez les conditions générales
3. Cliquez sur **"Antrag einreichen"**
4. Vous recevrez un email de confirmation

## 👤 Mon Dashboard

### Accéder au Dashboard

1. Connectez-vous à votre compte
2. Cliquez sur votre nom en haut à droite
3. Sélectionnez **"Mein Dashboard"**

### Vue d'Ensemble

Le dashboard affiche :
- **Demandes en cours** : Statut de vos demandes
- **Prêts actifs** : Vos prêts en cours

## 🆘 Résolution de Problèmes

### Je ne reçois pas l'email de confirmation

1. Vérifiez votre dossier spam
2. Vérifiez l'adresse email saisie
3. Attendez 10 minutes
4. Contactez le support si toujours rien

### Je ne peux pas me connecter

1. Vérifiez votre email et mot de passe
2. Utilisez "Mot de passe oublié"
3. Videz le cache de votre navigateur
4. Essayez un autre navigateur

### Mon document ne se télécharge pas

1. Vérifiez le format (PDF uniquement)
2. Vérifiez la taille (max 5MB)
3. Essayez de compresser le PDF
4. Contactez le support

### Ma demande est bloquée

1. Vérifiez les documents manquants
2. Consultez vos notifications
3. Contactez votre conseiller
4. Appelez le support
